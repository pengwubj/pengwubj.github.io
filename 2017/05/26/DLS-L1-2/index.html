System aspect of deep learning

  1. Faster Training
  2. Efficient serving
  3. Low memory consumpiton

Ingredient in Deep Learning

  1. Model and architecture
  2. Objective function , training techniques
    1. which feedback should we use to guide the algorithm
    2. Supervised , RL , adversarial training
  3. Regularization, initialization
  4. Data

Evolution of Convnets

  1. LeNet 1998
    • Basic structures : conv max-pooling, softmax
  2. Alexnet 2012
    • Relu Dropout
  3. GoogLeNet 2014
    • Multi-independent pass way ( Sparse weight matrix )
  4. Inception BN 2015
    • Batch normalization
  5. Residual net 2015
    • Residual pass way
    • Add transformation result to input, partly solve vanishing/explosive value problem

Challenges : From LeNet to AlexNet

  • Need much more data
    • ImageNet
  • A lot more computation burdens
    • GPU
  • Overfitting Prevention
    • Dropout regularization
  • Stable initialization and training
    • Explosive/Vanishing gradient problems
    • Require cares full tuning of initializaion and data normalization

Batch Normalization : Stabilize the Magnitude

  • Output is invariant to input scale
    • Easy to tune learning rate
    • Less sensitive initialization
  • Deep nets training was initialization sensitive
